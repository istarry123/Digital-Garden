用一台二手服务器开始 HomeLab 之旅。先后搭建了：

- Proxmox VE 虚拟化平台
- pfSense 软路由与 VLAN 隔离
- Windows Server AD 域控
- Ubuntu Server 运行 Docker 服务

HomeLab 成了最好的学习环境——可以随意折腾、重启、重装，不怕搞坏生产环境。
